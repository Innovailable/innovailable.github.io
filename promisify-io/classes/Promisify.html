Promisify Class

Defined in: src/promisify.coffee:4

Base class for promise based IO devices.

The type of the data which is sent and received is implementation specific. You will most probably use Buffer or String.

Implementation of actual devices have to use and overwrite the protected methods which start with an underscore.
