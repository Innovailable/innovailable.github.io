PromiseSocket Class

Extends Promisify
Defined in: src/socket.coffee:6

A promise wrapper around network sockets. It receives Buffer and can send Buffer and String.

If you want to connect to a server use PromiseSocketClient and if you want to create a server use PromiseSocketServer.
